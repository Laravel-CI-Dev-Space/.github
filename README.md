# Laravel Côte d'Ivoire 🇨🇮

> **Ensemble, construisons l'avenir du développement web ivoirien.**

Laravel Côte d'Ivoire est la première communauté structurée dédiée aux développeurs Laravel en Côte d'Ivoire et dans la diaspora ivoirienne.

On est plus de **500 membres** — juniors, confirmés, freelances, salariés, étudiants — réunis autour d'un même framework et d'une même conviction : les développeurs ivoiriens méritent un espace à eux pour apprendre, partager, progresser et peser dans l'écosystème tech mondial.

---

## Pourquoi Laravel Côte d'Ivoire ?

Laravel Côte d'Ivoire est née d'une conviction simple : **la Côte d'Ivoire a les talents. Il lui manquait juste l'espace pour les réunir.**

Pendant longtemps, les développeurs Laravel ivoiriens ont évolué sans communauté locale. Pas d'espace structuré, pas d'événements dédiés, pas de réseau commun. Chacun apprenait dans son coin, sans pairs, sans mentors, sans visibilité.

On a décidé que ça devait changer.

On croit que les prochaines grandes applications africaines seront construites par des développeurs ivoiriens. On croit que ces développeurs méritent les mêmes ressources, le même réseau et la même visibilité que leurs pairs en Europe ou en Amérique du Nord. Et on croit que ça commence ici — entre nous, maintenant.

---

## Notre mission

- **Partager** — Produire et diffuser des ressources Laravel de qualité en français, adaptées au contexte ivoirien
- **Entraider** — Créer un espace où chaque développeur, quel que soit son niveau, peut poser ses questions et trouver de l'aide
- **Connecter** — Relier les développeurs ivoiriens aux opportunités locales et internationales
- **Contribuer** — Initier et maintenir des projets open source utiles à l'écosystème Laravel africain

## Comment contribuer ?

Toutes les contributions sont les bienvenues — que tu sois junior ou senior, développeur ou rédacteur.

### Tu peux contribuer en :

- ⭐ Mettant une étoile sur les projets qui t'intéressent
- 🐛 Signalant des bugs via les Issues
- 💡 Proposant des améliorations via les Pull Requests
- 📝 Rédigeant ou améliorant la documentation
- 🌍 Traduisant des ressources en français
- 💬 Répondant aux questions dans les Discussions

### Pour commencer

```bash
# 1. Fork le projet qui t'intéresse
# 2. Clone ton fork
git clone https://github.com/ton-pseudo/nom-du-projet.git

# 3. Crée une branche pour ta contribution
git checkout -b feature/ma-contribution

# 4. Fais tes modifications et commit
git commit -m "feat: description de ma contribution"

# 5. Push et ouvre une Pull Request
git push origin feature/ma-contribution
```

Consulte le fichier `CONTRIBUTING.md` de chaque projet pour les instructions spécifiques.

## Code de conduite

Laravel Côte d'Ivoire s'engage à maintenir un espace bienveillant et inclusif pour tous ses membres.

### Ce qu'on attend de chaque membre

- Utiliser un langage accueillant et respectueux
- Accepter les critiques constructives avec ouverture
- Se concentrer sur ce qui est meilleur pour la communauté
- Faire preuve d'empathie envers les autres membres

### Ce qu'on n'accepte pas

- Harcèlement sous toutes ses formes
- Commentaires insultants, dégradants ou discriminatoires
- Spam, publicité non sollicitée, autopromotion excessive
- Tout comportement inapproprié qui nuit à l'ambiance communautaire

Tout manquement à ces règles peut entraîner une exclusion temporaire ou définitive des espaces communautaires.
